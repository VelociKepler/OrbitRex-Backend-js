# Prerequisites
- NPM
- MONGODB
- CLOUDINARY (upcoming...)
- STRIPE (upcoming...)

# Get start project with git clone this repo
```
git clone git@github.com:VelociKepler/OrbitRex-Backend-js.git
```
using normal way if u dont put your ssh key with github
# install package
```
cd OrbitRex-Backend-js
```
and
```
npm install
```

# Start backend service with
```
nodemon server.js
```
enjoy!
